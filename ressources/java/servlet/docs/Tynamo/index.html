Tynamo™-TINI Documentation
[Go to the Tynamo™ Home Page]

This server is a conditionally compliant HTTP/1.1 server and Servlet 2.2 servlet container for the TINI platform. The goal is to provide both speed and conformance to the specifications.

The main entry point is the class com.qindesign.tini.http.Main. A single argument that specifies the location of the server configuration properties file is required. Details about this file are found here.
